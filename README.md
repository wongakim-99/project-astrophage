# 프로젝트명 : Project-Astrophage

## 1. 핵심 컨셉 : 지식의 성간 항해 (Interstellar Knowledge Trip)

단순한 기록이 아닌, 내 머릿속 뉴런 네트워크(우주 거대 구조)를 직접 탐사하며 지식 간의 연결고리를 육안으로 확인한다.

> **잊혀진 지식은 암흑 물질이 되고, 연결된 지식은 항성이 된다**
> 

인간의 뉴런 구조 == 우주 구조와 유사하다는 철학 하에, 내 머릿속 뉴런에 있는 지식을 우주 구조로 묘사하여 성간 여행하며 지식을 탐색하는 개인 지식 블로그.

## 3. 항성 생애주기 시스템 (지식 상태)

실제 천체물리학 항성 진화 사이클을 기반으로 한다.

> 성운 → 주계열성 → 적색 거성 → 백색 왜성 → 흑색 왜성
> 

| 단계 | 조건 | 시각 표현 |
| --- | --- | --- |
| 주계열성 (Active) | 30일 내 3회 이상 조회 | 밝은 청백색, 크게 |
| 황색 왜성 (Normal) | 30일 내 1~2회 조회 | 노란빛, 보통 크기 |
| 적색 거성 (Fading) | 60~90일 미조회 | 붉고 흐릿하게 팽창 |
| 백색 왜성 (Forgotten) | 90일+ 미조회 | 희미한 흰 점 |
| 암흑 물질 (Lost) | 180일+ 미조회 | 거의 안 보임 |

### 부활 시스템

- **직접 복습** → 강한 에너지 회복 (체류 시간 + 방문 횟수 반영)
- **연관 개념 복습 → 신성(Nova) 효과** → 약한 에너지 전파

### 신성(Nova) 효과 — 연관 복습 에너지 전파

백색 왜성이 동반성에서 물질을 흡수해 일시적으로 밝아지는 실제 천문 현상에서 착안.

- A 개념 복습 시, 웜홀 또는 중력으로 연결된 B, C 개념에 에너지 소량 전파
- 연관도(유사도)가 높을수록 전파량 많음
- 직접 복습 에너지의 20~30% 수준으로 제한 → 연관 복습만으로는 주계열성 복귀 불가
- 시각 연출 : 신성 효과 발생 시 연결된 항성들이 잠깐 반짝이는 애니메이션

## 4. 분류 시스템 — 웜홀로 해소

CS 지식은 트리가 아닌 그래프 구조이기 때문에 완벽한 분류 강제는 불가능하다.

**해결 방식**

- 항성은 하나의 **주 은하**에 귀속
- 보조 태그로 연관 은하 추가 지정 가능
- 다른 은하의 항성과 유사도가 임계값 이상이면 **자동 웜홀 생성**
- 분류 변경 시 주 은하만 수정, 비용 없음

**웜홀 규칙**

- 물리학적 정의 : 공간적으로 멀리 떨어진 두 지점을 연결하는 지름길
- 항성은 한 곳에만 존재하되, 연결 통로(웜홀)만 존재
- 웜홀은 유사도 임계값 이상일 때만 생성 → 시각적 복잡도 제어

---

## 5. 의미 유사도 계산

**방식 : Embedding API → 코사인 유사도 → 거리 변환**

- 개념 추가 시 백엔드에서 자동 임베딩 계산
- 같은 은하 내 항성 간 거리 = 유사도 기반 자동 배치
- 웜홀 생성 기준 = 다른 은하 항성과 유사도가 임계값 초과 시
- 중력은 거리 제곱에 반비례 → 유사도 높을수록 인력 강해 가까이 배치 (물리학적으로 타당)

## 6. UI 렌더링 전략

렌더링 지옥 방지를 위해 **줌 레벨 3단계로 씬 분리**

| 레벨 | 보이는 것 | 렌더링 부하 |
| --- | --- | --- |
| 은하단 뷰 | 은하들만, 항성 미표시 | 매우 낮음 |
| 은하 뷰 | 해당 은하 항성 + 웜홀 | 중간 |
| 항성 뷰 | 개념 상세 + 인접 항성 | 낮음 |

**최적화 전략**

- LOD (Level of Detail) : 멀리 있는 항성은 점으로만, 줌인 시 이름/상세 렌더링
- 엣지 culling : 유사도 임계값 이하 연결선 미표시
- 은하 단위 lazy load : 클릭해서 들어갈 때 해당 은하 항성 로드
- Three.js WebGL 기반 렌더링

## 7. 사용자 개입 — 항성 생성 플로우

- 항성 추가 플로우
    
    개념 이름 입력 → 마크다운 설명 작성 → 주 은하 선택 (기존 or 신규) → 자동 임베딩 계산 → 유사 항성 미리보기 → 확인 후 우주에 배치
    
- 새 은하 생성도 자유롭게 가능 → **우주가 팽창하는 메타포**
- 항성 추가 연출 : "항성 탄생" 애니메이션
- 은하 간 이동, 재분류 비용 없음 → 애자일하게 운영

## 8. 블로그 성격

- 항성 상세 페이지 = 외부 공유 가능한 마크다운 블로그 포스트
- 우주 탐색 뷰는 나만의 탐색 인터페이스

**철학이 살아있는 UX 디테일**

- 진입 시 우주 전체 맵에서 항성으로 날아가는 인트로 애니메이션
- 오랫동안 안 본 개념 = 점점 어두워지는 시각 피드백
- 웜홀 통과 = 실제 이동하는 느낌의 트랜지션 효과
